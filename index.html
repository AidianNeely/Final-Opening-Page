Overview Of Computer Science Class:
This class had personally taught me to do things I've only ever seen the final results of. For example, being able to create websites like this in a matter of a few days is something I used to never imagine being able to do. Now I know how to make websites, basic games, and other cool things with code.
The main compiler we used this year was Open Proccessing. Open Proccessing is a compiler accessiable from pretty much any device you can imagine. The requirements are a search engine and an account. By having these to things you can create and view amazing and creative projects made by others or yourself. So I would reccomend going and getting an account and starting your own project.
We also covered different counting bases. Such as base 2 (binary), base 3 (trinary), and base 16 (hexadecimal). We eventually came to the end of the semester and started learning about web programming. I instantly knew I wanted to do my final on this and here I am. We used github as our compiler for our HTML (Hypertext Markup Language), CSS (Cascading Style Sheets), and JavaScript code.
